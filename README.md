# DragonGP.Ai

AI-powered stock analysis platform with real-time technical indicators, market data, and probability predictions.

## Features

- 🤖 **AI Stock Analysis** - Technical indicators (RSI, MACD, Trend Analysis)
- 🎯 **AI Mark 6** - Lottery probability predictions
- 💳 **Stripe Integration** - One-time and subscription payments
- 🎤 **Voice Controls** - Voice input and text-to-speech
- 🌐 **Multi-language** - English, Cantonese (廣東話), Mandarin (普通話)
- 📊 **PDF Reports** - Download analysis reports

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **UI:** Tailwind CSS, shadcn/ui
- **Backend:** Supabase
- **Payments:** Stripe
- **Deployment:** Vercel
